The Dragon Tera Shard (Japanese: テラピースドラゴン Tera Piece Dragon) is a type of item introduced in Generation IX. It is a type of Tera Shard, which can be used to change the Tera Type of a Pokémon. It corresponds to the Dragon type.

Effect

50 Dragon Tera Shards can be used to change a Pokémon's Tera Type to Dragon at the Treasure Eatery in Medali, except for Ogerpon and Terapagos.
